Toppval bland kurser vid Mittuniversitetet, HT2024

Inför höstterminen 2024 fanns det hela 377 kurser att söka vid Mittuniversitetet och det var många sökande, flerta med över tusen kandidater faktiskt, så konkurrensen var stenhård. De allra mest populära kurserna tycks ifjol ha handlat om bland annat kriminologi och psykologi. Se diagrammet nedan för toppvalen i siffror.

Toppval bland program vid Mittuniversitetet, HT2024

Inför höstterminen 2024 fanns det även 131 program att söka vid Mittuniversitetet och det var många sökande även här, återigen några med över tusen kandidater, men med ett program extra utstickande. Hela 32% av de sökande, eller 2343 personer försökte ifjol komma in på Webbutvecklingsprogrammet! (varav 578 hade det som förstahandsval). Se diagrammet nedan för vilka övriga program som stack ut i popularitet.
